主页 · 改版说明

① 风格：浅色 × 浅蓝

主背景 #f3f7fc，卡片纯白 + 1px 浅蓝描边，主色 #1f6feb，辅助色 #3aa0ff。整体清爽通透，符合移动端阅读习惯。

② 信息层级（自上而下）

  1. 顶部状态：HELLO ANCHOR + 设备在线 / 离线 胶囊（绿/灰）
  2. 设备主卡：型号 + 切换按钮 / 设备图 / 聚合状态 + 运行/温度/电量（去掉别名）
  3. 双卡信号：移动 + 电信 dBm 与强度条
  4. 实时速率：上下行大数字（去掉折线和切换 Tab）+ LIVE 心跳标记
  5. WiFi 卡片：SSID + 密码（默认 ●●●●，点眼睛切换明文）
  6. 套餐卡片：流量套餐 / 聚合套餐 剩余条 + 已用 / 有效期

③ 关键交互

  • 点击 切换设备 ⇆ → 从底部弹出已绑定设备抽屉，列表 + 底部 + 绑定新设备
  • WiFi 密码 👁 眼睛图标：单击切换 ●●●● ↔ 明文
  • 套餐进度条颜色：流量蓝、聚合深蓝；剩余少于 30% 自动转橙提示（可后续接入）
  • 速率数字每秒滚动刷新（保留），不再绘制曲线

④ 待你确认

  • 浅蓝色调饱和度是否合适，需要更淡还是更鲜艳？
  • 设备切换抽屉是否符合预期？是否需要加入"删除/解绑"按钮？
  • 套餐卡片是否需要显示价格或续费提醒？
