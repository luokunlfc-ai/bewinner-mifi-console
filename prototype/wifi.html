WiFi 管理 · 设计说明

① WiFi 信息卡

  • 顶部 WiFi 信号动画 + "WiFi 已开启"状态
  • SSID / 密码（眼睛切换） / 安全类型 / 频段
  • 底部「WiFi 设置」按钮 → 弹出设置抽屉（与主页共用）

② 已连接设备列表

  • 标题右侧显示已连接数量
  • 每行：设备图标 + 名称 + IP + 连接时长 + 断开按钮
  • 设备图标按类型区分：手机 / 笔记本 / 摄像机 / 平板
  • 点击设备行 → 弹出设备详情抽屉（MAC / 信号 / 已用流量等）
  • 点击「断开」→ 二次确认弹窗 → 确认后移除设备行并更新计数

③ 关键交互

  • 密码切换：与主页一致的眼睛图标交互
  • 设备详情：点击设备名称区域弹出底部抽屉
  • 断开设备：点击断开按钮 → 确认弹窗 → 设备行淡出移除
  • WiFi 设置：底部抽屉含 SSID / 密码 / 安全类型 / 频段 / 隐藏SSID
